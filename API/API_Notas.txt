Ejecutar una instancia de .NET
> dotnet new console

Necesario instalar los nugets:
> dotnet add package Microsoft.EntityFrameworkCore.SqlServer
> dotnet add package Microsoft.EntityFrameworkCore.Design
> dotnet add package Microsoft.EntityFrameworkCore.Tools
> dotnet add package Microsoft.VisualStudio.Web.CodeGeneration.Design
> dotnet add package Microsoft.AspNetCore.Mvc.NewtonsoftJson


Obtener el scaffold db
> dotnet ef dbcontext scaffold "Server=tcp:universidadamericana-sql.database.windows.net,1433;Initial Catalog=ExamenII;Persist Security Info=False;User ID=sa-universidadamericana-sql;Password=UAM2021.;MultipleActiveResultSets=False;Encrypt=True;TrustServerCertificate=False;Connection Timeout=30;" Microsoft.EntityFrameworkCore.SqlServer -o Models
> String no esta encriptada

Migraciones de database
> No necesarias. No modificaciones a la base de datos

Para los servicios se pueden basar en este
> https://docs.microsoft.com/en-us/aspnet/core/tutorials/first-web-api?view=aspnetcore-5.0&tabs=visual-studio
> Tener en cuenta de NoContent(), ActionResult(), NotFound() y otros no funcionan en API porque no lo heredan
> Se pueden probar en el swagger de NETCORE
